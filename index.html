Quiniela web con ~100 usuarios diarios, construida con Next.js, Supabase y alojada en Vercel. Disene la arquitectura de datos y orqueste agentes de IA para la generacion del codigo. def invertir_lista(cabeza):
    anterior = None
    actual = cabeza
    while actual:
        siguiente = actual.siguiente
        actual.siguiente = anterior
        anterior = actual
        actual = siguiente
    return anterior
