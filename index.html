Next.js Supabase Vercel

Fiebre Mundialista

AI-Assisted Architect & Data Orchestrator

Quiniela web con ~100 usuarios diarios, construida con Next.js, Supabase y alojada en Vercel. Disene la arquitectura de datos y orqueste agentes de IA para la generacion del codigo. Los administradores gestionan puntuaciones y resultados; los usuarios consultan su quiniela y exploran las de los demas en tiempo real.

El Desafio

Soportar ~100 usuarios diarios con consultas concurrentes, diferenciando roles de admin y usuario sin comprometer rendimiento.

La Solucion

Supabase con Row Level Security para roles, Vercel para hosting, y orquestacion de IA para acelerar el desarrollo full-stack.
